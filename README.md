# HPRSSN ReaPack Repository

REAPER scripts and extensions by HYPRESSION.

## Installation

1. In REAPER: Extensions → ReaPack → Import repositories
2. Paste URL: `https://github.com/hypressi0n/hprssn-reapack/raw/main/index.xml`
3. Extensions → ReaPack → Browse packages → Install

## Available Packages

- **HPRSSN Hidden VKB** — REAPER/Ableton Layout, Exclusive Arm & Solo, etc
